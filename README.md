# Historias Clínicas - Base de Datos

Base de datos desarrollada en SQL Server para la gestión de:

- Pacientes
- Médicos
- Consultas
- Estudios clínicos

## Características
- Modelo relacional
- Claves primarias y foráneas
- Integridad referencial

## Tecnologías
- SQL Server

## Cómo usar

1. Abrir SQL Server Management Studio
2. Ejecutar el script completo (F5)
3. La base se crea automáticamente
